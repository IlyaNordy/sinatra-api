# sinatra-api

Учебный API-проект на Sinatra.

## О проекте
Небольшой backend-проект, созданный для практики работы с Sinatra, построения API и взаимодействия с PostgreSQL.

## Стек
- Ruby
- Sinatra
- Sinatra Contrib
- Sinatra Namespace
- Sequel
- PostgreSQL
- JSON
- Puma
- Dotenv

## Что реализовано
- базовая структура API-приложения
- настройка Sinatra-сервиса
- подготовка к работе с PostgreSQL через Sequel
- обработка JSON-данных
- запуск приложения через Rack/Puma

## Цель проекта
Изучение лёгкого Ruby-фреймворка Sinatra и практики создания backend/API-приложений без Rails.

## Запуск проекта
```bash
bundle install
ruby app.rb
